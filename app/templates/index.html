Garbage Classifier

A deep learning model that identifies an object as cardboard, glass, metal, plastic, paper, or trash. 
Inspired by the global landfill crisis and growing human ignorance.

Start Classifying

Project Overview

This garbage classifier was created to address the recycling crisis.  The goal of the AI was to be able to eliminate confusion regarding which category everyday wastes belong to. Our product can successfully determine if an object is cardboard, glass, metal, plastic, paper, or trash. In the future, AIs such as this classifier may be able to aid humans in making seemingly-minor but meaningful decisions that impact the environment and society.

WORKFLOW

Kaggle

Data Collection: our dataset of pictures of trash was obtained from Kaggle. It consisted of 1200+ photos! There were 6 different categories of trash: cardboard, plastic, glass, paper, metal, & general trash.

Kaggle dataset

Labelbox

Data Preparation: after pulling all our data from Kaggle, Labelbox was used to identify which category of garbage each image belonged to.

labelbox

Yolo

Training: YOLO is an object detection system which uses a pre-built algorithm to detect objects in photos and identify what they are. We trained the algorithm using our labeled Kaggle data.

yolo

Evaluation

Deployment: After the model was evaluated and tested on new data, it was deployed to this website for everyone to use as a final product!
